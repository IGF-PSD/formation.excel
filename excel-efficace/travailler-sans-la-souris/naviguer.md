# Naviguer

Il est plus efficace de se déplacer dans un feuillet Excel exclusivement au clavier :

* les flèches directionnelles ⇨⇦⇧⇩ permettent de passer d’une case à l’autre ;
* associées à la touche CRTL, les flèches directionnelles permettent de se déplacer au sein d’un bloc de cellules vide ou pleines ;
* associées à la touche SHIFT, elles permettent de sélectionner plusieurs cellules à la fois ;
* passer d’un feuillet à l’autre avec les touches CRTL + haut de page ou CRTH + bas de page.

{% hint style="info" %}
_Pour se déplacer rapidement dans une plage non encore remplie mais où un copier-coller doit être effectué, placer un « x » dans la dernière case de la plage, de manière à la sélectionner facilement grâce à CRTL + SHIFT + flèche._
{% endhint %}

Sélectionner, ajouter et supprimer des lignes et des colonnes :

* sélectionner une colonne : SHIFT + barre d’espace ;
* sélectionner une ligne : CRTL + barre d’espace ;
* ajouter une ligne : CRTL + « + » ;
* supprimer une ligne : CRTL + « - » ;
