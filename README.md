# Formation Excel

## Objectifs de la formation

Deux formations sont proposées : Excel niveau 1 et niveau 2.

Excel niveau 1 vise à remplir les objectifs suivants :

* se familiariser avec les commandes de bases d'Excel ;
* être en capacité de réaliser des graphiques avec la charte graphique du Service ;
* réaliser des manipulations simples sur une base de données volumineuse : tri, filtre ou mise en forme conditionnelle notamment ;
* savoir réaliser un tableau croisé dynamique (TCD).

Excel niveau 2 vise à remplir les objectifs suivants :

* être à l'aise avec l'utilisation de fonctions et leur imbrication ;
* être en capacité de réaliser des analyses statistiques simples : distribution d'une variable, étude de corrélations ;
* savoir utiliser les fonctions RechercheV, Equiv et Index ;
* savoir réaliser une régression linéaire.

## Public cible

Excel 1 est destiné à l'ensemble des nouveaux arrivants et convient à un public débutant. Il est nécessaire d'avoir suivi Excel 1 ou d'être à l'aise avec Excel pour suivre la formation Excel 2.
